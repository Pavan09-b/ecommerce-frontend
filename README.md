# 🛒 Ekart - Frontend eCommerce Application

Ekart is a responsive and dynamic eCommerce web application built using modern frontend technologies.  
The application simulates a real-world online shopping experience with product listings, cart management, and a clean user interface.

---

## 📌 Features

- 🛍️ Product listing page
- 🔍 Product details view
- 🛒 Add to Cart functionality
- 🧮 Cart quantity management
- 💰 Dynamic price calculation
- 📱 Fully responsive design
- 🎨 Styled using Tailwind CSS utility classes
- ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- Vite
- Tailwind CSS
- ESLint
